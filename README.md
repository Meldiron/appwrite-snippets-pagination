# 👩‍💻 Appwrite Snippet - Get all documents

## 📜 Requirements

- Node.JS & NPM
- TypeScript
- Git

## 🛠 Setup

- `git clone` this repo
- `cp .env.example .env`, and fill with your credentials
- `npm i` to install dependencies

## 🚀 Running

- `tsc` to build the project
- `node dist/index.js` to run the script
